## Description

The ASCII85 (base85) decoder and encoder
```
$ ascii85    # читает байты из STDIN и выдаёт ASCII85 в STDOUT
$ ascii85 -e # то же самое
$ ascii85 -d # читает ASCII85 из STDIN и выдаёт байты в STDOUT; при ощбке завершается с код
```
